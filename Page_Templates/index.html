Object Control

  • Manage—Bulk operations on a collection with multiple items
  • Single-Page Edit—Form collecting information through a small number of fields to update an existing record
  • Long Form Edit—Form collecting information through a large number of fields to update an existing record, using a panel to navigate multiple sections
  • Multi-Page Panel Edit—Long form distributed across multiple pages navigated from a sidebar panel, collecting information to update an existing record
  • Single-Page Inspect—A read-only display of record with a small number of attributes and related data

Task Initiation

  • Advanced Select (Lightbox)—A dialog to search for a record to return to a text input field in the main view's form.
  • Simple Confirm (Lightbox)—A dialog spawned after a process has been requested, but before it is initiated, to communicate the consequences of that action.

    • Faceted Search—A simple search whose results are filtered by selecting from attributes contained within that data.
    • Inspect & Perform—Review read-only record data before initiating an action on that record to change its state.
    • Interactive Review—An editable form allowing the scope or configuartion of a requested action to be changed or canceled before proceeding to initiate that process.

    • Start—A collection of information, quick functions and navigation that describe the key uses of a group of views.
    • Composite Search—A navigational search for an object leveraging a single text area for input by selecting one of multiple search options in a dropdown.
